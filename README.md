# Analise_de_credito
Sistema de Análise de Crédito

Este projeto consiste em uma aplicação web simples desenvolvida em Python utilizando a biblioteca Streamlit, com o objetivo de simular um processo básico de análise de crédito. A aplicação permite que o usuário insira informações como nome, idade, renda mensal e score de crédito e, com base em regras previamente definidas no código, o sistema retorna automaticamente se o crédito é aprovado ou reprovado.

O desenvolvimento deste projeto teve como foco a prática de lógica de programação aplicada a um cenário financeiro real. Durante a construção da aplicação, foram utilizadas estruturas condicionais (if, elif, else), entrada e manipulação de dados fornecidos pelo usuário e definição de regras de negócio traduzidas em lógica programável. A interface foi criada com Streamlit, permitindo interação simples e direta com o usuário por meio de uma aplicação web.

As decisões de aprovação são baseadas em critérios como idade mínima, renda mínima e score mínimo, que podem ser facilmente ajustados para simular diferentes políticas de crédito. O projeto demonstra como decisões que parecem complexas no contexto financeiro podem ser estruturadas em regras claras dentro de um sistema.

Para executar o projeto, basta clonar o repositório, instalar as dependências listadas no arquivo requirements.txt e executar o comando streamlit run seguido do nome do arquivo principal da aplicação.

Este projeto representa um passo importante na consolidação dos fundamentos de programação em Python e na aplicação prática de lógica voltada para tomada de decisão, servindo como base para evoluções futuras, como implementação de validações mais robustas, persistência de dados ou expansão das regras de análise.
